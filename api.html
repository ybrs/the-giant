API¶

This part of the documentation covers all the interfaces of The-Giant.

Main Interface¶

Mainly you need to create an application (or a function) and give it to thegiant.server.run method

thegiant.server.listen(application, host, port) → None¶

Makes thegiant listen to host:port and use application as WSGI callback. (This does not run the server mainloop.)

thegiant.server.run(application, host, port) → None¶

Calls listen(application, host, port) and starts the server mainloop.

run() -> None
Starts the server mainloop. listen(...) has to be called before calling run() without arguments.

Timers¶

thegiant.server.add_timer()¶

Add a timer

thegiant.server.stop_timer()¶

Stop a timer

thegiant.server.restart_timer()¶

Restart a timer

Utilities¶

These helper functions are used to format response values easily

thegiant.helpers.reply(v)¶

formats the value as a redis reply

The-Giant is a redis protocol toolkit, think it like wsgi-ish server that speaks Redis.
