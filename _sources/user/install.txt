.. _install:

Installation
============

This part of the documentation covers the installation of Requests.
The first step to using any software package is getting it properly installed.


Distribute & Pip
----------------

Installing the-giant is simple with `pip <http://www.pip-installer.org/>`_::

    $ pip install thegiant


Get the Code
------------

The-Giant is actively developed on GitHub, where the code is
`always available <https://github.com/ybrs/the-giant>`_.

You can either clone the public repository::

    git clone git://github.com/ybrs/the-giant.git

Once you have a copy of the source, you can embed it in your Python package,
or install it into your site-packages easily::

    $ python setup.py install
